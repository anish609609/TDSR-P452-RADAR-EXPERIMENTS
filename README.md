# TDSR-P452-RADAR-EXPERIMENTS

A hands-on exploration using the TDSR P452 radar to learn by building, testing, and questioning real data. The focus is on understanding systems from the ground up, validating ideas through experiments, and developing intuition by working with real-world signals rather than relying only on theory or simulations.

## Getting Started

The repository includes breathing datasets acquired using the radar system. Users can run the provided MATLAB scripts on these files to understand how raw radar data behaves in practice.

- Navigate to the breathing dataset folder  
- Run the provided MATLAB scripts  
- Select any dataset file when prompted and observe the results  

## Exploration

The provided pipeline is not restrictive. It serves only as a starting point for understanding the data.

Users are encouraged to explore beyond it by trying different approaches such as windowing techniques, filtering methods, and phase-based estimation for improved analysis.

## Philosophy

The aim is to move beyond theoretical understanding and develop real intuition by working directly with measured data. Every dataset is an opportunity to question assumptions, test ideas, and refine methods.

## Contribution

Feel free to experiment, improve the processing pipeline, and contribute better approaches or insights. The goal is to collaboratively build a deeper understanding of radar signal processing using real-world data.
